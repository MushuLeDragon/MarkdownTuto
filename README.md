## Accentuation
```
Accentuation Italique : *asterisks* or _underscores_.

Accentuation Gras : **asterisks** or __underscores__.

Accentuation Combinée : **asterisks and _underscores_**.

Accentuation Barrée avec deux tildes : ~~Scratch this.~~
```

## Listes
